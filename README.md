# Productivity-Dashboard# Productivity Dashboard

A modern, responsive **Single Page Application (SPA)** built using **HTML5, CSS3, and Vanilla JavaScript**. This dashboard brings together essential productivity tools into one clean interface, helping users stay organized and focused throughout the day.

The project was built to strengthen my understanding of frontend development by implementing real-world functionality instead of relying on frameworks or copy-pasted code.

---

## Features

* ⏱️ **Pomodoro Timer**

  * Start, pause, and reset functionality
  * Helps maintain focused work sessions

* ✅ **Task Manager**

  * Add and delete tasks
  * Mark tasks as completed
  * Highlight important tasks
  * Persistent task storage using Local Storage

* 📅 **Hourly Scheduler**

  * Plan your day hour by hour
  * Automatically saves schedules
  * Data persists after page refresh

* 🌤️ **Weather Integration**

  * Displays current weather information
  * Fetches live data using a Weather API

* 💡 **Motivational Section**

  * Displays motivational content to keep users engaged

* 🌙 **Dark / Light Mode**

  * Theme switching using CSS Custom Properties
  * Smooth user experience without page reloads

* 📱 **Responsive Design**

  * Optimized for desktop, tablet, and mobile devices
  * Adaptive layouts using Flexbox and Media Queries

* ✨ **Modern UI**

  * Interactive hover effects
  * Mouse-tracking spotlight animation
  * Smooth transitions and micro-interactions
